About

This Website can be used to plot mathematical equations.

Key Features

  1. Diverse equations
    • GraphJS handles standard arithmetic operations, including addition, subtraction, multiplication, and division. The equations have to be in a syntax that can be executed as PHP code.
    • Some functions such as square root (sqrt()), exponential (x ** 2) and trigonometric calculations (sin(a), cos(a), tan(a)) are available.
  2. Graph plotting
    • Users can input a mathematical equation into the calculator.
    • The equation will be executed and as a result you will see the plot of that equation
  3. Constants
    • GraphJS supports two important constants: PI and E
    • These can be used as follow: %pi, %e (e.g. sin(%pi * x))
  4. Saving equations
    • You can save your equation by checking the "Save equation" checkbox and by giving it a name.
    • Saved equations will be listed in a dropdown, so you can load a previous equation
