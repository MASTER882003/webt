About

This Website can be used to perform numerical calculations and graphical representation of mathematical functions.

Key Features

  1. Numerical Calculations
    • The calculator handles standard arithmetic operations, including addition, subtraction, multiplication, and division. The calculations have to be in a syntax that can be executed as PHP code.
    • Advanced functions such as square root (sqrt()), exponentiation (x^2) and trigonometric calculations (sin(a), cos(a), tan(a)) are available.
  2. Graph plotting
    • Users can input mathematical functions, both simple and complex, into the calculator.
    • Plotting can be done by using the plot(equation) function
  3. Constants
    • The calculator can use some important constants like PI or E.
    • List of all available constants: %pi, %e
  4. Saving equations
    • You can save your equation by checking the "Save equation" checkbox and by giving it a name.
